## Collect a performance recording from Edge Developer tools

1. Open Microsoft Edge, then open **Developer tools** by pressing **F12**.

2. Select the **Performance** tab, select **Clear**, then select **Record**. reproduce the issue.

    ![Start recording](/assets/images/edgeperf1.png)

3. **Reproduce** the issue. Select **Stop** after it's reproduced.

    ![Stop recording](/assets/images/edgeperf2.png)

4. Select **Save trace**

    ![Save trace](/assets/images/edgeperf3.png)

> For more information, refer to: [https://learn.microsoft.com/en-us/microsoft-edge/devtools-guide-chromium/evaluate-performance/?source=recommendations#record-runtime-performance](https://learn.microsoft.com/en-us/microsoft-edge/devtools-guide-chromium/evaluate-performance/?source=recommendations#record-runtime-performance)
