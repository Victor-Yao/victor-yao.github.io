## Collect Console log from Developer tools on Edge

1. Open Microsoft Edge, then open **Developer tools** by pressing **F12**.

2. Select the **Console** tab, then select **Clear console**.

   ![Edge DevTools Console with Clear console option](/assets/images/edgeconsole1.png)

3. Reproduce the issue.

4. Right-click in the Console output, then select **Save as...**.

   ![Save as option for Edge DevTools Console output](/assets/images/edgeconsole2.png)
