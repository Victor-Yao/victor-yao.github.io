## Collect offline diagnostic data from Edge on iOS

1. Open Microsoft Edge, go to `edge://flags`, then enable **Microsoft Edge for Business debug mode**. Close and reopen Edge.

    ![Enable Microsoft Edge for Business debug mode](/assets/images/edgeios-diagnostic1.jpg)

2. Reproduce the issue.

3. Open **Help and feedback**:

   1. In **Normal** mode, tap the user profile (upper-left corner).

        ![Normal-help and feedback ](/assets/images/edgeios-diagnostic2.jpg)

   2. In **InPrivate** mode, tap the more settings (lower-right corner).
        
        ![private-Open help and feedback 01](/assets/images/edgeios-diagnostic3.png)

        ![private-Open help and feedback 02](/assets/images/edgeios-diagnostic4.png)

3. Turn on all options, then tap **Share**.

    ![Check all options](/assets/images/edgeios-diagnostic5.jpg)

4. Save the diagnostic data file locally.

    ![Save file locally](/assets/images/edgeios-diagnostic6.jpg)
