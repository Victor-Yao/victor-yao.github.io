## Export Application and System windows event log.

1. Search for **Event Viewer**, then open it.

2. In the left pane, right-click **Application** (or **System**), then select **Save All Events As...**.

   ![Save All Events As option in Event Viewer](/assets/images/windows-event1.png)

3. Save the file with the `.evtx` extension.

   ![Save event log as an .evtx file](/assets/images/windows-event2.png)
