## Capture a network trace with Network Monitor (NetMon)

1. Download and install [Microsoft Network Monitor (NetMon)](https://www.microsoft.com/en-us/download/details.aspx?id=4865) on both the client and the server.

   > Perform the remaining steps on the client and server at the same time.

2. Open NetMon, then select **New Capture**.

   ![New Capture option in NetMon](/assets/images/netmon1.jpg)

3. Select **Start**.

   ![Start capture button in NetMon](/assets/images/netmon2.jpg)

4. Reproduce the issue.

5. Select **Stop**.

   ![Stop capture button in NetMon](/assets/images/netmon3.jpg)

6. Save the trace files.

   ![Save trace option in NetMon](/assets/images/netmon4.jpg)
